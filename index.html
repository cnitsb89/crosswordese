How this works

Answer mode trains clue→answer recall: type the answer and press Enter. Pattern mode hides one letter of a common answer so you learn letter-shape statistics; type the missing letter. Grading is automatic: a wrong answer comes back this session, a correct one is spaced out. If you took longer than the “slow” cutoff, it asks Easy vs Hard so slow-but-right cards return sooner.

Answers are the recurring NYT fill appearing 30+ times across 2009–2021, ranked by frequency so you learn the highest-value words first. Progress saves in this browser only. Keyboard: Enter check / next, 1 Hard, 2 Easy when prompted.
